# GM Motors NY

Premium pre-owned / high-line dealership website. Static HTML + Bootstrap 5.3 (CDN), no build step.

## Pages
- `index.html` — Homepage
- `inventory.html` — Inventory listing (search/filter + grid)
- `vehicle.html` — Vehicle detail page (VDP)

## Develop
```
python3 -m http.server 8124
# → http://localhost:8124/
```

See `CLAUDE.md` for design direction and conventions.
